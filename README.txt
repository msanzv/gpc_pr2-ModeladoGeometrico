
Opciones de teclado:


-Ver2Curvas.exe:

	-Las teclas 1-4 corresponden a la Curva 1, y las teclas 5-8. 
	Seleccionando un punto y un eje (x,y,z), se pueden usar las 
	teclas ARRIBA/ABAJO para mover el punto por la coordenada. 
	-Tecla 'a': Iniciar/detener animación
	-Tecla 'p': Mostrar/Ocultar polígonos característicos
	-Tecla 'q': Salir


-VerSuperficie.exe:

	-Tecla 'a': Iniciar/detener animación
	-Tecla 'm': Mostrar/Ocultar polígonos característicos
	-Tecla 'q': Salir
	


Nota: 	Ambos ejecutables tienen un menú secundario (botón derecho) desde
	el cual se pueden realizar estas y otras acciones.
